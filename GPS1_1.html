Pair Programming

What is Pair Programming?

Pair programming is an interactive learning experience where individuals work with a partner with a focus toward completing a specific goal. Pair programming allows for a diversity of ideas, techniques, and experiences.

Navigator

  • The Navigator talks and leads the driver.
  • The Navigator describes what code they are looking at by the line number.
  • The Driver presses the gas, but the Navigator says when to turn.

Driver

  • The Driver gives input, but should be focused on the area of code instructed by the navigator.
  • The Driver doesn't look at a line of code they are not told to look at by the Navigator.
  • The Driver doesn't type code or comments before discussing with the Navigator.

Software Engineers pair in order to:

  • Reduce code errors and/or catch errors early.
  • Share or transfer knowledge.
  • On-board new developers.
